## 📘 How to Use the Web App

1. **Download and Open the Project**  
   Download the project and open it using **Visual Studio Code**.

2. **Start the Application**  
   Start the project. After a few seconds, a **Swagger** page should automatically open in your browser.

3. **Open the Web Interface (UI)**  
   In a new browser tab, enter the following address:

       http://localhost:PORT/wwwroot/index.html

   Replace `PORT` with **your specific port number**, which you’ll see in the terminal when the project starts (e.g., `5000` or `7167`).
